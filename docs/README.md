## 📝 文档编写规范

所有文档应遵循以下规范：

1. **清晰的标题结构** - 使用 Markdown 标题层级
2. **目录导航** - 长文档应包含目录
3. **代码示例** - 提供完整可运行的代码示例
4. **截图说明** - 必要时添加截图辅助说明
5. **更新日期** - 在文档末尾注明最后更新日期

## 🔄 文档维护

- 新增功能时，应同步更新相关文档
- 修复 Bug 时，应记录修复说明
- 定期审查和更新过时的文档
