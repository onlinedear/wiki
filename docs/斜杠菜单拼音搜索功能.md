# 斜杠菜单拼音搜索功能

## 功能说明

在编辑器中输入 `/` 唤起斜杠菜单后，现在支持以下搜索方式：

### 1. 中文搜索
直接输入中文关键词，例如：
- 输入 `有序` → 匹配 "有序列表"
- 输入 `标题` → 匹配 "一级标题"、"二级标题"、"三级标题"

### 2. 拼音全拼搜索
输入完整拼音，例如：
- 输入 `youxu` → 匹配 "有序列表"
- 输入 `biaoti` → 匹配所有标题项
- 输入 `wuxu` → 匹配 "无序列表"

### 3. 拼音首字母缩写搜索
输入拼音首字母，例如：
- 输入 `yx` → 匹配 "有序列表"
- 输入 `bt` → 匹配所有标题项
- 输入 `wx` → 匹配 "无序列表"
- 输入 `gt` → 匹配 "甘特图"

### 4. 英文搜索（原有功能）
输入英文关键词，例如：
- 输入 `numbered` → 匹配 "Numbered list"
- 输入 `heading` → 匹配所有标题项

## 技术实现

### 依赖库
使用 `pinyin-pro` 库进行拼音转换：
- 全拼转换：`pinyin(text, { toneType: "none", type: "array" })`
- 首字母转换：`pinyin(text, { pattern: "first", toneType: "none", type: "array" })`

### 匹配逻辑
1. 对翻译后的中文文本进行拼音转换
2. 同时生成全拼和首字母缩写
3. 使用模糊匹配算法（fuzzyMatch）进行匹配
4. 支持部分匹配，例如输入 `you` 可以匹配 `youxu`

### 修改文件
- `apps/client/src/features/editor/components/slash-menu/menu-items.ts`
  - 添加 `pinyin-pro` 导入
  - 新增 `matchesPinyin` 辅助函数
  - 在 `getSuggestionItems` 中集成拼音匹配逻辑

## 性能考虑

- 拼音转换在每次搜索时实时进行
- 使用 try-catch 包裹拼音转换，避免异常影响搜索
- 模糊匹配算法时间复杂度为 O(n)，性能良好
