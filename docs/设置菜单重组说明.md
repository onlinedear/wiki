# 设置菜单重组说明

## 概述

重新组织了设置菜单的分类结构，使其更加清晰和符合逻辑。

## 新的菜单结构

### 账户 (Account)
个人账户相关设置：
- 个人资料 (Profile)
- 偏好设置 (Preferences)
- API 密钥 (API keys)
- 第三方系统集成 (Third-party integrations)

### 用户 (Users)
用户管理相关（仅管理员可见）：
- 用户 (Members)
- 用户组 (Groups)

### 内容 (Content)
内容管理相关（仅管理员可见）：
- 文档库 (Spaces)
- 公开分享 (Public sharing)
- 评论管理 (Comment management)

### 系统 (System)
系统级设置（仅管理员可见）：
- 站点设置 (General)
- 安全与单点登录 (Security & SSO)
- API 管理 (API management)
- 邮件服务 (Mail service) - 仅自托管版本
- 账单 (Billing) - 仅云版本
- 许可证与版本 (License & Edition)

## 主要改进

1. **更清晰的分类**：将设置项按照功能和用户角色进行了重新分组
2. **权限控制**：非管理员用户只能看到"账户"分类
3. **逻辑分离**：
   - 个人设置与系统设置分离
   - 用户管理独立成组
   - 内容管理独立成组
   - 系统配置集中管理
