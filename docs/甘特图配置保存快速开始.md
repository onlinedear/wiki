# 甘特图配置保存 - 快速开始

## 5 分钟上手指南

### 步骤 1：设置你的个性化视图

根据需要设置筛选、排序或分组：

**示例：查看进行中的高优先级任务**

1. 点击"筛选"按钮
2. 添加条件：任务状态 = 进行中
3. 添加条件：优先级 = 高
4. 点击"排序"按钮
5. 添加条件：截止日期（升序）
6. 点击"分组"按钮
7. 选择：负责人

### 步骤 2：观察保存按钮

- 工具栏上的"保存配置"按钮（软盘图标）变为绿色
- 这表示有未保存的更改

### 步骤 3：保存配置

- 点击绿色的"保存配置"按钮
- 按钮变回灰色并禁用
- 配置已保存！

### 步骤 4：验证保存

- 刷新浏览器页面（F5 或 Ctrl+R / Cmd+R）
- 甘特图自动恢复你的配置：
  - ✅ 筛选条件已应用
  - ✅ 排序规则已生效
  - ✅ 分组显示正常

## 常见使用场景

### 场景 1：每日站会视图

**目标**：快速查看今天和明天要完成的任务

```
1. 筛选：截止日期 ≤ 明天
2. 排序：优先级（高到低）
3. 分组：负责人
4. 保存配置
```

**效果**：每天打开文档，立即看到需要关注的任务。

### 场景 2：项目进度视图

**目标**：了解项目各阶段的进展

```
1. 分组：任务状态
2. 排序：开始日期（升序）
3. 保存配置
```

**效果**：清晰看到未开始、进行中、已完成的任务分布。

### 场景 3：个人任务视图

**目标**：只看自己负责的任务

```
1. 筛选：负责人 = 我
2. 排序：截止日期（升序）
3. 保存配置
```

**效果**：专注于自己的工作，不被其他任务干扰。

### 场景 4：紧急任务视图

**目标**：优先处理紧急且重要的任务

```
1. 筛选：优先级 = 高
2. 筛选：任务状态 ≠ 已完成
3. 排序：截止日期（升序）
4. 保存配置
```

**效果**：快速识别需要立即处理的任务。

## 实用技巧

### 技巧 1：先设置后保存

不要每次修改都保存，完成所有设置后再点击"保存配置"，这样可以：
- 减少不必要的保存操作
- 避免保存中间状态
- 提高性能

### 技巧 2：利用按钮颜色

- **灰色**：当前视图已保存，可以放心刷新
- **绿色**：有未保存的更改，刷新会丢失

### 技巧 3：临时视图

如果只是临时查看某个视图，不要保存：
1. 修改筛选/排序/分组
2. 查看结果
3. 刷新页面恢复原配置

### 技巧 4：团队协作

作为团队负责人：
1. 设置团队统一的视图
2. 保存配置
3. 通知团队成员刷新页面

### 技巧 5：多视图管理

如果需要多个固定视图：
1. 复制甘特图块（Ctrl+C / Cmd+C）
2. 粘贴到文档其他位置
3. 为每个甘特图设置不同的配置
4. 分别保存

例如：
- 甘特图 1：按状态分组（项目进度视图）
- 甘特图 2：按负责人分组（人员分配视图）
- 甘特图 3：按优先级排序（优先级视图）

## 注意事项

### ⚠️ 配置会影响所有协作者

保存的配置对所有查看文档的人可见。如果你是团队成员而非负责人，修改配置前请先与团队沟通。

### ⚠️ 删除字段前清除配置

如果某个字段被用于筛选、排序或分组，删除该字段前应该：
1. 清除相关的筛选条件
2. 清除相关的排序条件
3. 清除分组（如果使用该字段）
4. 保存配置
5. 再删除字段

### ⚠️ 文档需要保存

配置保存到甘特图节点后，还需要等待文档自动保存（或手动保存）才能真正持久化到数据库。

### ⚠️ 折叠状态不保存

分组的折叠/展开状态不会被保存，每次加载时所有分组都是展开的。

## 故障排除

### 问题：刷新后配置没有恢复

**检查清单：**
1. ✓ 是否点击了"保存配置"按钮？
2. ✓ 按钮是否变回灰色？
3. ✓ 文档是否已保存？（查看标题栏是否有未保存标记）
4. ✓ 是否使用了强制刷新？（尝试 Ctrl+Shift+R / Cmd+Shift+R）

### 问题：其他协作者看不到我的配置

**可能原因：**
1. 他们没有刷新页面
2. 文档还没有同步（等待几秒后刷新）
3. 他们在你保存前就打开了文档（让他们刷新）

### 问题：配置保存后想撤销

**解决方法：**
1. 使用文档撤销功能（Ctrl+Z / Cmd+Z）
2. 或者重新设置配置并保存

### 问题：按钮一直是绿色

**可能原因：**
1. 你修改了配置但没有保存
2. 点击"保存配置"按钮即可
