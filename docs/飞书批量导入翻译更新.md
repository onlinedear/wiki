## 使用示例

### 代码中使用翻译

```typescript
// 标签页标题
{t('Single Document')}  // 单文档
{t('Batch Import')}     // 批量导入

// 按钮文本
{t('Load Documents')}   // 加载文档
{t('Import Selected Documents')} ({count})  // 导入选中的文档 (N)

// 提示信息
{t('Enter a Feishu folder token to browse and select multiple documents for import.')}
// 输入飞书文件夹 Token 以浏览和选择多个文档进行导入。

// 错误消息
{t('No documents found')}  // 未找到文档
{t('Failed to load documents')}  // 加载文档失败
```

### 通知消息

```typescript
notifications.show({
  title: t('Import completed'),  // 导入完成
  message: t('Documents imported successfully'),  // 文档导入成功
  color: 'teal',
});

notifications.show({
  title: t('No documents selected'),  // 未选择文档
  message: t('Please select at least one document to import'),  // 请至少选择一个文档进行导入
  color: 'yellow',
});
```

## 注意事项

### 1. 翻译文件格式
- JSON格式，UTF-8编码
- 键值对形式
- 注意逗号和引号

### 2. 翻译一致性
- 保持术语统一（如"文档"、"导入"等）
- 与现有翻译风格一致
- 简洁明了，易于理解

### 3. 特殊字符
- 使用中文标点符号
- 保留必要的英文术语（如Token、URL）
- 数字和计数使用阿拉伯数字
