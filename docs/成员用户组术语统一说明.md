# 成员和群组术语统一说明

## 修改概述

根据用户需求，将中文翻译中的"成员"统一改为"用户"，"群组"统一改为"用户组"，以提供更一致、更清晰的术语表达。

## 修改原因

1. **术语准确性**: "用户"比"成员"更准确地表达了这个概念
2. **避免歧义**: "成员"可能与"会员"混淆，"用户"更明确
3. **术语一致性**: "用户组"比"群组"更符合中文习惯，与"用户"形成统一的命名体系

## 修改统计

共修改了 **40+** 处翻译，涵盖以下类别：

### 1. 用户相关（20+处）

#### 基础术语
- Member → 用户
- Members → 用户
- member → 用户
- members → 用户

#### 操作相关
- Add members → 添加用户
- Invite members → 邀请用户
- Invite new members → 邀请新用户
- Manage members → 管理用户
- Delete member → 删除用户
- Remove space member → 移除文档库用户
- Remove group member → 移除用户组用户
- Add space members → 添加文档库用户
- Add group members → 添加用户组用户

#### 提示消息
- Members added successfully → 用户添加成功
- Member removed successfully → 用户移除成功
- Member deleted successfully → 用户删除成功
- Member role updated successfully → 用户角色更新成功
- Invited members who are yet to accept → 尚未接受邀请的用户将显示在这里

#### 确认对话框
- Are you sure you want to delete this workspace member? → 您确定要删除此工作区用户吗？
- Invited members will be granted access → 被邀请的用户将被授予访问权限

#### 权限描述
- Can become members of groups and spaces → 可以成为工作区中用户组和文档库的用户
- Select role to assign to all invited members → 选择要分配给所有被邀请用户的角色
- Once enforced, all members must enable → 一旦实施，所有用户必须启用双因素认证

### 2. 用户组相关（20+处）

#### 基础术语
- Group → 用户组
- Groups → 用户组
- Group name → 用户组名称
- Group description → 用户组描述

#### 操作相关
- Create group → 创建用户组
- Delete group → 删除用户组
- Edit group → 编辑用户组
- Manage Group → 管理用户组
- Add groups → 添加用户组
- Add to groups → 添加到用户组
- Search for groups → 搜索用户组
- Search for users and groups → 搜索用户和用户组
- No group found → 未找到用户组

#### 确认对话框
- Are you sure you want to delete this group? → 您确定要删除这个用户组吗？
- Members will lose access to resources this group has access to → 用户将失去对该用户组可访问资源的访问权限
- Are you sure you want to remove this user from the group? → 您确定要从用户组中移除这个用户吗？

#### 示例文本
- e.g Group for developers → 例如：开发人员用户组

## 完整的术语体系

修改后的完整术语体系：

```
站点 (Workspace)
  ├── 用户 (User/Member)
  ├── 用户组 (Group)
  │   └── 用户组用户
  └── 文档库 (Space)
      ├── 文档库用户
      └── 文档 (Document)
          └── 子文档 (Subpage)
```

## 注意事项

1. **浏览器缓存**: 修改后需要清除浏览器缓存或硬刷新
2. **术语一致性**: 确保在所有地方都使用"用户"和"用户组"
3. **上下文理解**: 
   - "用户" = 系统中的个人账户
   - "用户组" = 用户的集合，用于权限管理

## 用户体验提升

### 1. 术语清晰
"用户"比"成员"更清晰，避免了与"会员"等概念的混淆。

### 2. 命名一致
"用户组"与"用户"形成统一的命名体系，更容易理解。

### 3. 符合习惯
"用户"和"用户组"是中文用户熟悉的概念，降低了学习成本。

## 总结

本次修改将所有的"成员"统一改为"用户"，"群组"统一改为"用户组"，共修改40+处翻译。

修改后的术语体系：
- ✓ Member/Members → 用户
- ✓ Group/Groups → 用户组
- ✓ Group members → 用户组用户
- ✓ Space members → 文档库用户

这样的命名更加清晰明确，符合中文用户的使用习惯，提升了整体的用户体验。
