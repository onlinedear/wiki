# 邮件服务配置 - 快速开始

## 5 分钟快速配置

### 步骤 1: 访问邮件服务设置

1. 以**管理员**身份登录 NoteDoc
2. 点击右上角头像 → **设置**
3. 在左侧菜单找到 **系统** → **邮件服务**

### 步骤 2: 配置 SMTP 信息

根据您的邮件服务商填写以下信息：

#### 使用 Gmail（推荐用于测试）

```
SMTP 主机: smtp.gmail.com
SMTP 端口: 587
SSL/TLS: ❌ 关闭
忽略 TLS: ❌ 关闭
SMTP 用户名: your-email@gmail.com
SMTP 密码: [应用专用密码]
发件人邮箱: your-email@gmail.com
发件人名称: NoteDoc
```

> **获取 Gmail 应用专用密码**:
> 1. 访问 https://myaccount.google.com/security
> 2. 启用两步验证
> 3. 搜索"应用专用密码"
> 4. 生成新的应用专用密码

#### 使用 QQ 邮箱

```
SMTP 主机: smtp.qq.com
SMTP 端口: 587
SSL/TLS: ❌ 关闭
SMTP 用户名: your-email@qq.com
SMTP 密码: [授权码]
发件人邮箱: your-email@qq.com
发件人名称: NoteDoc
```

> **获取 QQ 邮箱授权码**:
> 1. 登录 QQ 邮箱
> 2. 设置 → 账户
> 3. 开启 SMTP 服务
> 4. 生成授权码

### 步骤 3: 保存配置

点击页面底部的 **保存设置** 按钮

### 步骤 4: 测试配置

1. 在"测试邮件"部分输入您的邮箱地址
2. 点击 **发送测试邮件**
3. 检查邮箱（包括垃圾邮件文件夹）
4. 如果收到测试邮件，配置成功！✅

## 常见问题

### ❓ 测试邮件发送失败

**可能原因**:
1. SMTP 用户名或密码错误
2. 端口被防火墙封锁
3. 邮件服务商需要启用 SMTP 服务
4. 需要使用应用专用密码而非账户密码

**解决方法**:
- 检查用户名和密码是否正确
- 尝试不同的端口（587, 465, 25）
- 确认邮件服务商已启用 SMTP
- 使用应用专用密码或授权码

### ❓ 收不到邮件

**检查清单**:
- ✅ 检查垃圾邮件文件夹
- ✅ 确认发件人邮箱地址正确
- ✅ 检查邮件服务商的发送限制
- ✅ 查看服务器日志

### ❓ 端口选择

| 端口 | 加密方式 | 推荐 | 说明 |
|------|---------|------|------|
| 25   | 无/STARTTLS | ❌ | 常被 ISP 封锁 |
| 465  | SSL/TLS | ✅ | 需要开启 SSL/TLS |
| 587  | STARTTLS | ✅✅ | 最推荐 |

## 验证配置

配置完成后，您可以通过以下方式验证：

1. **邀请新用户**: 邀请一个新用户，检查是否收到邀请邮件
2. **密码重置**: 尝试重置密码，检查是否收到重置邮件
3. **通知邮件**: 触发通知，检查是否收到通知邮件
