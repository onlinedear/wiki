## 🎓 学习路径

### 初级用户（1-2 小时）

1. 阅读[甘特图功能说明](./甘特图功能说明.md)的前半部分
2. 实践：插入甘特图，添加几个任务
3. 实践：切换视图模式，拖拽调整任务时间
4. 阅读[甘特图配置保存快速开始](./甘特图配置保存快速开始.md)
5. 实践：设置筛选条件并保存

### 中级用户（2-3 小时）

1. 阅读[甘特图字段配置和设置功能](./甘特图字段配置和设置功能.md)
2. 实践：创建自定义字段，调整字段显示
3. 阅读[甘特图筛选功能说明](./甘特图筛选功能说明.md)
4. 实践：设置复杂的筛选条件
5. 阅读[甘特图排序功能说明](./甘特图排序功能说明.md)
6. 实践：设置多条件排序

### 高级用户（3-4 小时）

1. 阅读[甘特图分组功能说明](./甘特图分组功能说明.md)
2. 阅读[甘特图分组快速开始](./甘特图分组快速开始.md)
3. 实践：按不同字段分组，体验折叠/展开
4. 阅读[甘特图配置保存功能说明](./甘特图配置保存功能说明.md)
5. 实践：创建多个甘特图，每个保存不同的配置
6. 探索：结合筛选、排序、分组创建复杂视图

### 开发者（4-6 小时）

1. 阅读所有用户文档，了解功能全貌
2. 阅读[甘特图排序实现要点](./甘特图排序实现要点.md)
3. 阅读[甘特图配置保存实现总结](./甘特图配置保存实现总结.md)
4. 查看源代码：
   - `packages/editor-ext/src/lib/gantt/gantt.ts`
   - `apps/client/src/features/editor/components/gantt/gantt-view.tsx`
5. 阅读[甘特图排序测试清单](./甘特图排序测试清单.md)
6. 实践：运行测试，验证功能
