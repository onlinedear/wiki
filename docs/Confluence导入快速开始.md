# Confluence 导入快速开始

## 🚀 5 分钟快速上手

### 方式一：在线导入（推荐）

#### 第一步：获取 Confluence Token

1. 访问您的 Confluence 实例
2. 点击头像 → Settings → Personal Access Tokens
3. 创建新 Token（需要读取权限）
4. 复制生成的 Token

#### 第二步：配置 NoteDoc

1. 登录 NoteDoc
2. 进入 **账户设置** → **我的资料**
3. 找到 **Confluence Integration** 部分
4. 填写：
   - Confluence URL: `https://your-domain.atlassian.net/wiki`
   - Personal Access Token: 粘贴刚才复制的 Token
5. 点击 **测试连接** 验证
6. 点击 **保存配置**

#### 第三步：导入文档

1. 打开目标文档库
2. 点击 **导入文档**
3. 选择 **Confluence** → **从在线导入**
4. 粘贴 Confluence 文档 URL
5. 选择是否包含子文档
6. 点击 **从 Confluence 导入**

✅ 完成！文档将自动导入到 NoteDoc

---

### 方式二：ZIP 文件导入

#### 第一步：从 Confluence 导出

1. 在 Confluence 中打开要导出的文档
2. 点击 **...** → **Export to HTML**
3. 选择导出选项（包含子页面、附件等）
4. 下载 ZIP 文件

#### 第二步：导入到 NoteDoc

1. 打开目标文档库
2. 点击 **导入文档**
3. 选择 **Confluence** → **上传 ZIP 导出**
4. 选择下载的 ZIP 文件

✅ 完成！

## 💡 使用技巧

### 获取文档 URL

**方法 1：从浏览器地址栏复制**
```
https://your-domain.atlassian.net/wiki/pages/123456/Page+Title
```

**方法 2：从分享链接获取**
1. 在 Confluence 中打开文档
2. 点击 **Share** 按钮
3. 复制链接

### 批量导入

如果要导入整个文档空间：

1. 找到空间的根文档
2. 启用 **包含子文档** 选项
3. 导入将递归处理所有子文档

### 测试导入

建议先测试：

1. 选择一个简单的文档
2. 不包含子文档
3. 验证导入效果
4. 确认无误后再批量导入

## ⚠️ 注意事项

1. **Token 权限**：确保 Token 有读取权限
2. **网络连接**：在线导入需要稳定的网络连接
3. **导入时间**：大量文档可能需要较长时间
4. **浏览器标签**：导入过程中不要关闭标签页
5. **文档权限**：只能导入您有权限访问的文档

## 🔧 故障排除

### 连接失败

**问题**：测试连接失败

**解决方案**：
- 检查 URL 格式是否正确
- 确认 Token 未过期
- 验证网络连接

### 无法提取文档 ID

**问题**：提示无法从 URL 提取文档 ID

**解决方案**：
- 使用完整的文档 URL
- 避免使用短链接
- 从浏览器地址栏复制 URL

### 导入失败

**问题**：导入过程中出错

**解决方案**：
- 确认有文档访问权限
- 尝试导入单个文档测试
- 检查错误消息详情
- 联系管理员
